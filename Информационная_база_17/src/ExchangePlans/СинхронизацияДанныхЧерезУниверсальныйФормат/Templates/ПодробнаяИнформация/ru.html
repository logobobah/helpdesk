Синхронизация данных между корреспондентами производится на основе универсального формата данных.

Более подробная информация о методике настройки и использования этого вида синхронизации данных представлена в описании совместного использования на сайте ИТС.

В базу СУЗ данные из других баз не загружаются. Из конфигурации СУЗ выгружаются следующие объекты:


Объект базы СУЗ Объект формата Описание
Документ.Трудозатраты Документ.РеализацияТоваровУслуг Документы к обмену регистрируются при следующих условиях:
1) Документ проведен;
2) Состояние не равно "Новый" или "Отменен";
3) ИНН организации заказчика заполнен.
В документе все работы суммируются по проектам. В табличную часть реализации попадает описание услуги и название проекта, как при формировании печатной формы "Акт об оказании услуг".
Справочник.Организации Справочник.Организации К обмену регистрируются элементы справочника с установленными флагами "Актуален", "Собственная организация".
Справочник.Заказчики Справочник.Контрагенты К обмену регистрируются элементы справочника с установленным флагом "Актуален" и заполненным ИНН организации заказчика.
Справочник.Организации Справочник.Номенклатура Выгружается по заполненному полю "Описание выполняемой услуги" в карточке организации. Выгружается только по ссылке.
Справочник.Организации Справочник.ЕдиницыИзмерения Выгружается по заполненному полю "Ед. изм. услуги" в карточке организации. Выгружается только по ссылке.
Справочник.Организации Справочник.Договоры Выгружается по заполненному полю "Договор основание" в карточке организации. Выгружается только по ссылке. Название договора должно быть в формате "Договор № <Номер> от <Дата>".
Справочник.Организации Справочник.БанковскиеСчета Выгружаются только для выгрузки собственных организаций по заполненным банковским реквизитам в карточке организации.
Справочник.Организации Справочник.Банки Выгружаются только для выгрузки банковских счетов собственных организаций по заполненным банковским реквизитам в карточке организации.
Справочник.Организации Справочник.Валюты Всегда выгружются только рубли.
Документ.Трудозатраты Справочник.Файлы В файл выгружается печатная форма "Трудозатраты". Владельцем файла является объект формата Документ.РеализацияТоваровУслуг.

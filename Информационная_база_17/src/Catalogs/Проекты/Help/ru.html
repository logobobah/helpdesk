Справочник "Проекты".

  • Предназначен для классификации задач исполнителей по проектам. Используется в связке с заказчиком. Хранит следующую информацию: связь с заказчиком, куратора со стороны заказчика, приоритет, классификацию и описание. 
  • На основании проекта есть возможность ввести задачу, написать письмо, установить напоминание.
  • Из карточки проекта есть возможность просмотреть все задачи по проекту, прикрепить к нему файлы, обсудить его на форуме.

Справочник "Записи календаря исполнителя".

Календарь исполнителя предназначен для планирования времени. В нём можно вводить различные записи и отмечать время, доступное для планирования. Для создания новой записи календаря необходимо выделить нужное время мышкой и отпустить ее.

Напоминания

Для записей календаря можно устанавливать напоминания. Для этого необходимо включить настройку программы Использовать напоминания. После этого станет доступна настройка напоминаний в карточке записи календаря. 

Настройка повторения записей календаря
Для записей календаря можно указывать настройки повторения. После этого запись с заданными настройками повторения будет отображаться в календаре. Отдельный элемент повторения можно изменить или удалить. При изменении всех записей повторения данные в отдельных элементах повторения будут заменены на новые общие настройки.

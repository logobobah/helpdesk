Справочник "Условия выполнения".

Содержит в себе список условий, по которым в сценариях выполняются команды. Условие можно настроить двумя способами:
1) в режиме конструктора настроить фильтр для объектов;
2) через произвольный программный код.
При проверке условия из сценария система будет проверять оба режима.

Описание переменных программного кода.

Общие переменные для выполняемого кода.

1) Сценарий, тип Ссылка на выполняемый сценарий.
2) Результат, тип Булево. Если установить значение Ложь, то команды выполняться не будут.
3) ПараметрыСценария, тип Структура. Для передачи произвольных данных от условий сценария к командам. 

Переменные для вида события При изменении источника.

1) Источник, тип Объект типа указанного в поле "Тип источника", объект проверки.
2) ПараметрыСценария.ИсточникПередЗаписью, тип Структура из реквизитов и табличных частей источника до записи, для нового объекта Неопределено. Структура содержит следующие ключи: Реквизиты, ТабличныеЧасти, ДополнительныеСвойства, ИмяМетаданных, Тип.
3) ПараметрыСценария.ЭтоНовый, тип Булево, признак записи нового объекта в базу.
4) ПараметрыСценария.РежимЗаписи, тип РежимЗаписиДокумента, только при записи документа.
5) ПараметрыСценария.РежимПроведения, тип РежимПроведенияДокумента, только при записи документа.
6) Отказ, тип Булево, используется для отказа от записи при установленном виде события "При записи источника" .

Переменные для вида события Повторяющееся событие.

1) Источники, тип Массив со ссылками типа указанного в поле "Тип источника". Используется для повторяющегося события, чтобы выполнять команды на ними.

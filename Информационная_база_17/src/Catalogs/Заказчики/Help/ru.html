Справочник "Заказчики".

  • Предназначен для классификации задач исполнителей по заказчикам. Хранит следующую информацию: конакты, куратора со стороны заказчика, приоритет, классификацию, описание и юридические реквизиты. 
  • На основании заказчика есть возможность ввести задачу, написать письмо, установить напоминание.
  • Из карточки заказчика есть возможность просмотреть все задачи заказчика, все его проекты, контактных лиц заказчика, прикрепить к нему файлы, обсудить его на форуме, уставноить часовые ставки работы с ним.

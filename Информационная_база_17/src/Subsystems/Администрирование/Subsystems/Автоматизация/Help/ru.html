Подсистема "Автоматизация".

Преимущества подсистемы Автоматизация.

    1. Возможность перевести часть разработки из конфигурации в режим предприятия. Т.е. не нужно в дальнейшем "выгонять" всех пользователей из базы, блокировать ее, чтобы внести очередные правки в алгоритмы.
    2. Позволяет систематизировать внесенные изменения, объединять их в группы.
    3. Пользователи системы смогут часть необходимых изменений вносить без помощи программиста, а также видеть какие изменения применяются к различным объектам конфигурации.

Возможности подсистемы Автоматизация.

    1. Выполнение произвольного кода при записи объекта справочника и документа.
    2. Выполнение произвольного кода по расписанию и в фоновом задании.
    3. Выполнение кода из внешней обработки при записи объекта и по расписанию.
    4. Заполнение реквизитов объекта при записи объекта справочника и документа.
    5. Проверка на заполнение реквизитов объекта при записи с выводом настраиваемого сообщения.
    6. Оповещение по электронной почте по настраиваемому шаблону письма.
    7. Создание напоминания пользователям в самой базе.
    8. Создание задач исполнителям в базе.
    9. Включение/выключение существующих в базе сценариев.
